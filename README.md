# Portfolio

Personal portfolio website to showcase my skills, projects, experience, and contact links. Built with React, Vite, and Tailwind CSS.

## Tech Stack
- React 19
- Vite 7
- Tailwind CSS 4
- ESLint

## Features
- Responsive, fast, and lightweight
- Sections for About, Experience, Achievements, Projects, and Contact
- Content driven from simple data files in `src/data/`
- Easy to customize styling via Tailwind

## Getting Started

Prerequisites: Node.js 18+ and npm (or yarn/pnpm).

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```
Project run at http://localhost:3000
